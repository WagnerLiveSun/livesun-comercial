# LiveSun Comercial

Plataforma de gestao comercial e financeira em Flask, com suporte multiempresa, controle de acesso e operacao comercial integrada.

## Inicio rapido
1. Configure `.env` com `DB_NAME=comercial`.
2. Execute `iniciar_comercial.bat` no Windows ou `python run.py`.
3. Acesse `http://localhost:5000`.
